# online-doudizhu-pages

本仓库用于归档和发布多个独立 HTML 页面。这些页面彼此独立，不依赖特定后端服务，可以直接在浏览器中打开使用。

## 项目简介

online-doudizhu-pages 是一个纯前端的 HTML 页面集合仓库。所有页面均以静态文件形式存放，无需服务器环境即可运行。适合用于保存、分享或演示各类前端页面片段。

## 目录说明

```
online-doudizhu-pages/
├── pages/          # 存放各个独立 HTML 页面的目录
├── assets/         # 页面共用的资源文件（CSS、JS、图片等）
└── README.md       # 本文件
```

每个子文件夹或文件代表一个独立的功能页面，内部包含完整的 HTML 结构和必要资源。

## 页面归档说明

- 每个页面均以 `.html` 文件形式存在，可直接双击打开或部署到静态托管平台。
- 页面之间无耦合关系，可单独使用或修改。
- 部分页面可能引用 `assets/` 目录下的共用资源，请保持目录结构完整。

## 维护说明

- 欢迎提交新的独立页面或改进现有页面。
- 请确保每个页面在提交前能独立运行，不依赖外部 API 或在线资源（除非特别说明）。
- 新增页面请在 `pages/` 下创建独立文件夹，并更新本文件中的目录说明（可选）。
- 共用资源请统一放置在 `assets/` 目录下，避免重复引用。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 进入 `pages/` 目录，打开任意 `.html` 文件即可查看。
3. 也可将整个仓库部署到任意静态文件服务器（如 GitHub Pages、Netlify 等）后访问。

## 许可

本项目仅供学习和参考使用。所有页面内容均为独立创作或归档，不涉及任何特定网站或服务。
